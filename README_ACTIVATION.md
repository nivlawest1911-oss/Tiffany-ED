## üö® TROUBLESHOOTING

### **Common Issues**

#### **Database Error: "relation does not exist"**
**Solution:** Run `database/schema.sql` in Vercel Postgres Query tab

#### **Media Upload Fails**
**Solution:** 
1. Verify `BLOB_READ_WRITE_TOKEN` is set
2. Run: `vercel env pull .env.local`
3. Retry upload

#### **Google Login Fails**
**Solution:**
1. Go to Google Cloud Console
2. Add redirect URI: `https://edintel-app.vercel.app/api/auth/google/callback`
3. Wait 5 minutes and retry

#### **Build Fails**
**Solution:**
1. Run: `npm install`
2. Run: `npm run build`
3. Fix any TypeScript errors
4. Redeploy

**For more troubleshooting, see:** [VISUAL_SETUP_GUIDE.md](VISUAL_SETUP_GUIDE.md#-common-issues--solutions)
