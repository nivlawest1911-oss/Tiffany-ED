## üîß TROUBLESHOOTING

### **Issue: Logo Upload Fails**

**Solutions:**
1. Check file size (must be under 1MB)
2. Verify format (PNG or JPG only)
3. Ensure dimensions are 1024x1024px
4. Try different browser
5. Clear browser cache

### **Issue: Logo Appears Blurry**

**Solutions:**
1. Use the 1024x1024px version
2. Don't resize in browser
3. Upload original high-res file
4. Check DPI is 300

### **Issue: Logo Rejected by Google**

**Common Reasons:**
- Inappropriate content (our logo is fine)
- Low quality (our logo is high quality)
- Wrong dimensions (our logo is correct)
- Trademark issues (ensure you own rights)
